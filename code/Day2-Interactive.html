Numeric vs Integer Vectors

weights <- c(50, 60, 25, 35, 55)
class(weights)
[1] "numeric"
weights_Integer <- c(50L, 60L, 25L, 35L, 55L)
class(weights_Integer)
[1] "integer"

You can use the is._______ function to determine how the data is being stored by R

is.numeric(weights_Integer)
[1] TRUE

Similarly, you can use the as.________ function to redefine a set of data as a different kind of data. For example, R will read Zip Codes as numeric inputs. However, we know that Zip Codes are a category or a character data input.

zipcodes <- c(93105, 93514, 93101, 93401)
class(zipcodes)
[1] "numeric"
zipcodes_char <- as.character(zipcodes)
class(zipcodes_char)
[1] "character"
